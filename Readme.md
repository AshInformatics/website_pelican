## use

```
pip install pelican[Markdown]

pelican-quickstart
```

go through questions

```
pelican-themes --install Flex_angepasst/
pelican content -t Flex_angepasst -s pelicanconf.py



To check the website:
```
pelican --listen
```
then visit http://localhost:8000/
